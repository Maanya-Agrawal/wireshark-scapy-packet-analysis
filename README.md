# Packet Processing using Scapy and Wireshark

## Description

This project demonstrates packet capture and analysis using Wireshark and Scapy. Network traffic is captured and filtered based on different protocols such as TCP, UDP, TLS, ARP, and ICMP. The captured data is further processed using Scapy to extract useful statistics such as number of packets, unique IP addresses, and protocol distribution.

## Objectives

* To capture live network traffic using Wireshark
* To analyze packets using protocol-based filters
* To understand different layers of the TCP/IP model
* To process packet data using Scapy

## Technologies Used

* Wireshark
* Python (Scapy)

## How to Run

1. Capture network traffic using Wireshark and save the file (.pcapng)
2. Place the file in your working directory
3. Run the Python script:

```id="r8zv2k"
python3 script_name.py
```

## Conclusion

The experiment provides an understanding of packet-level analysis and protocol behavior. It demonstrates how tools like Wireshark and Scapy can be used together to analyze and process network traffic efficiently.
